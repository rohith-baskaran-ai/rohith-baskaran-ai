## Projects

### 🖼️ [CIFAR-10 Image Classification](https://github.com/rohith-baskaran-ai/cifar10-cnn-classifier)
Built a CNN from scratch in PyTorch. 4 conv blocks, batch normalization,
data augmentation. Reached **84.58% accuracy** on 10-class image classification.

### 💬 [Sentiment Analysis — LSTM](https://github.com/rohith-baskaran-ai/imdb-lstm-sentiment)
Built a Bidirectional LSTM from scratch — custom tokenizer, vocabulary of 25k words,
sequence padding. Trained on 25,000 IMDB reviews. **86.08% accuracy.**

### 🤖 [Sentiment Analysis — BERT](https://github.com/rohith-baskaran-ai/imdb-bert-sentiment)
Fine-tuned bert-base-uncased on the same IMDB dataset. **92.24% in 3 epochs.**
6% better than the LSTM with a fraction of the training time.
The gap between the two models shows exactly why transfer learning matters.
